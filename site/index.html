Built for care operations

Give every care worker the right workspace, not the whole system.

In home-care and assisted-living teams, overnight details can live across quick notes, family requests, preference changes, and shift checklists. Handoff Memory keeps those details organized by resident, then uses login, role checks, and assignments so each person sees only the patients they are responsible for.

Night caregiver Sees assigned residents and saves what changed
Morning care lead Opens assigned handoffs and asks follow-up questions
Supervisor Assigns shifts and keeps stale notes out

Access model

The app starts at sign-in, not at a shared role switcher.

The live product issues a JWT for the selected demo user. The backend uses that token to filter patients and reject actions outside the user's role. The public role preview below is for explanation; real workspaces are locked by authentication.

JWT session Every API request carries a signed role identity.
Assigned patients Caregivers and leads only receive their assigned residents.
Server-side RBAC Night staff cannot call handoff or supervisor cleanup APIs.
Supervisor control Shift assignment and memory cleanup live in the supervisor dashboard.

Memory layer

Cognee is the memory layer behind the handoff.

Care workers do not operate Cognee directly. The backend writes notes to Cognee, recalls resident-scoped context for handoffs, sends review feedback for improvement, and forgets stale notes before they mislead the next shift.

Caregiver saves a note remember
Morning lead asks for handoff recall
Supervisor marks a source important cognify
Team removes stale context forget
Why it matters

This is not a shared chat box with medical-looking notes.

Generic LLM workflow

Someone has to paste yesterday's context again, and everyone tends to see the same broad workspace.

Handoff Memory workflow

Users sign in, see assigned patients, and get source-backed memory only through their permitted workflow.

What the business gets

Faster handoffs, clearer accountability, patient-scoped recall, and safer cleanup of outdated notes.
