Persistent AI memory for high-stakes team handoffs.

ShiftMemory turns Cognee into the durable memory layer for shift-based teams. Workers add normal notes, supervisors review source-backed handoffs, and agents recall the same case memory across sessions.

Problem

Generic AI can summarize a shift. It cannot remember the next one.

Shift work depends on context that is scattered across notes, chats, files, and people's memory. The MVP uses caregiving as the demo wedge, but the architecture applies to support, logistics, field operations, incident response, and any workflow where handoff failures are expensive.

Cognee usage

Cognee is the memory substrate, not a checkbox integration.

Remember

Store shift notes, uploaded files, confirmed handoffs, task events, and supervisor decisions as case memory.

Recall

Retrieve scoped memory for generated handoffs, case Q&A, timeline reconstruction, and judge trace proof.

Improve

Feed supervisor corrections and importance signals back into memory so future handoffs get better.

Forget

Remove incorrect, stale, sensitive, or expired memories and verify they no longer appear in recall.
